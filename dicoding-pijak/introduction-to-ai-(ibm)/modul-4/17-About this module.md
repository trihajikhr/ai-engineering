# Is machine learning the answer to the unstructured data problem?

## Pendahuluan

Dalam modul ini, Anda akan mempelajari tentang **machine learning** (pembelajaran mesin) dan bagaimana ia bekerja menggunakan kalkulasi probabilitas.

## Tujuan Pembelajaran

Setelah menyelesaikan modul ini, Anda harus mampu:

- Mendefinisikan dan mendeskripsikan **machine learning**.
    
- Mendeskripsikan bagaimana **machine learning** menyusun **unstructured data**.
    
- Mendeskripsikan bagaimana **machine learning** menggunakan kalkulasi probabilitas untuk menyelesaikan masalah.
