# About this lesson: Introduction to generative AI

Model generatif telah digunakan bertahun-tahun dalam statistik untuk menganalisis data numerik. Kebangkitan _deep learning_ memungkinkannya menganalisis gambar, ucapan, dan tipe data kompleks lainnya. Namun, sebelum mendalami _generative AI_, penting untuk memahami AI secara lebih luas.

Dalam pelajaran ini, Anda akan mempelajari tentang:

- Kecerdasan buatan (_artificial intelligence_).
- Model _deep-learning_.
- Bagaimana model "belajar" untuk menghasilkan _output_ yang mungkin saat diberikan _prompt_.
